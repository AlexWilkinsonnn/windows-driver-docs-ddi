# NET_POWER_OFFLOAD_NS_PARAMETERS_INIT function


## -description

The **NET_POWER_OFFLOAD_NS_PARAMETERS_INIT** method initializes a [**NET_POWER_OFFLOAD_NS_PARAMETERS**](../netpoweroffload/ns-netpoweroffload-_net_power_offload_ns_parameters.md) structure.

## -parameters

### -param Parameters

A pointer to a client driver-allocated [**NET_POWER_OFFLOAD_NS_PARAMETERS**](../netpoweroffload/ns-netpoweroffload-_net_power_offload_ns_parameters.md) structure.

## -returns

This method does not return a value.

## -remarks

This method zeroes out the memory of the **NET_POWER_OFFLOAD_NS_PARAMETERS** structure, then fills in the **Size** member. Client drivers must then fill in the other members of the structure.
