# _NLSTABLEINFO structure


## -description


Stores the NLS file formats .


## -struct-fields




### -field OemTableInfo

Specifies OEM table.


### -field AnsiTableInfo

Specifies an ANSI table. 


### -field UpperCaseTable

Specifies an 844 format uppercase table.


### -field LowerCaseTable

 




#### - PUSHORT

Specifies an 844 format lowercase table.
