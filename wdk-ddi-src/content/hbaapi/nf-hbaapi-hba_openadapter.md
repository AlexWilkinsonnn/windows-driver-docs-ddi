# HBA_OpenAdapter function


## -description


The <b>HBA_OpenAdapter</b> routine opens an HBA and returns a handle.


## -parameters




### -param AdapterName

<p>Indicates the name of the HBA to open. </p>




## -returns



The <b>HBA_OpenAdapter</b> routine returns a handle to the open HBA.
