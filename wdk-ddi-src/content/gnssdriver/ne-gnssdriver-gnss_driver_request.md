# GNSS_DRIVER_REQUEST enumeration


## -description


GNSS_DRIVER_REQUEST enumerates  the GNSS driver data request types. 


## -enum-fields




### -field SUPL_CONFIG_DATA

This is used by the GNSS driver to  request SUPL configuration data.
